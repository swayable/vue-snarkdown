<h1 align="center">
  Vue Snarkdown
</h1>

A lightweight Vue component wrapping the tiny Markdown parser [Snarkdown](https://github.com/developit/snarkdown)

## Usage

_this_ is **easy** to `use`.

```html
<template>
  <vue-snarkdown>
    _this_ is **easy** to `use`.
  </vue-snarkdown>
</template>

<script>
import VueSnarkdown from './vue-snarkdown'

export default {
  components: {
    VueSnarkdown,
  },
}
</script>
```
